Установка:
Мод слід встановлювати на чисту гру - у папці з грою повинні 
знаходитись тільки файли та архіви гри і ніяких інших вкладених папок! 
dta архіви розпаковувати не потрібно! 
Установка стандартна - розпаковуємо архів "Bootlegger_Mod_v2.0.zip", 
будь-яким зручним способом, у папку з грою. Для правильної роботи 
модифікації в директорії гри повинні знаходитись такі файли та папки:

Bootlegger.exe
LS3D0.dll
rw_data.dll
MAPS
MISSIONS
DIFF
models
sounds
tables
anims

Для встановлення англомовної версії модифікації скористайтесь файлами
з теки "english_patch".
----------------------------------------------------------------------
Як грати:
Навіть не намагайтесь запускати гру з файлу "Game.exe" версії вищої за 
1.0, мод працюватиме неправильно! Щоб з цим не було проблем, в 
модифікації передбачений файл для запуску гри - "Bootlegger.exe". 
Запускаємо з нього гру, створюємо новий профіль і насолоджуємося!
